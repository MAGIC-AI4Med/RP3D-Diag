Codes in this folder can be used to train from scratch or finetune using a checkpoint

From scratch:
    Please set CHECKPOINT and SAFETENSOR as "None".
Finetune:
    please set CHECKPOINT as a checkpoint folder (can be derived from our huggingface) and SAFETENSOR to "None" .

'FUSE' and 'KE' represent our two strategy. Initially, they are set to late and False.
