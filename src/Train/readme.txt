Codes in this folder can be used to train from scratch or finetune using a checkpoint

From scratch:
    Please set CHECKPOINT and SAFETENSOR as "None".
Finetune:
    please set CHECKPOINT as a checkpoint folder (can be derived from our huggingface https://huggingface.co/QiaoyuZheng/RadDiag/tree/main/RP3D_5569_res_ART_256_32_BCE_T_4_2_0.7_2048_E_F_F_F_N) and SAFETENSOR to "None" .

'FUSE' and 'KE' represent our two strategy. Initially, they are set to late and False.
